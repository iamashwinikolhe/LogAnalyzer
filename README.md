# AI Log Analyzer (Local LLM)

## Features
- Chunking for large logs
- Embeddings using sentence-transformers
- FAISS vector search
- LLM analysis via Ollama
- Fully offline (no API key required)

## Run
## models llama3 , phi3
1. Start Ollama:
   ollama run llama3 

2. Install dependencies:
   pip install -r requirements.txt

3. Run:
   python analyzer.py
